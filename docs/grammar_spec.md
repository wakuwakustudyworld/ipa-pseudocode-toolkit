# IPA擬似言語 文法仕様

本ドキュメントは、IPA「試験で使用する情報技術に関する用語・プログラム言語など」Ver.5.1 の別紙1・別紙2に基づく擬似言語の文法をまとめたものである。

---

## 参照元

- IPA「試験で使用する情報技術に関する用語・プログラム言語など」Ver.5.1
- 別紙1: ITパスポート試験用
- 別紙2: 基本情報技術者試験，応用情報技術者試験用（Ver.5.1でAP追加）
- 2024年4月試験から適用

---

## 字句構造

### リテラル

```
整数リテラル   = [0-9]+
実数リテラル   = [0-9]+ "." [0-9]+
文字列リテラル = '"' 任意の文字列 '"'
論理リテラル   = "true" | "false"
未定義リテラル = "未定義" | "未定義の値"
```

### 識別子

```
識別子 = (英字 | "_" | 日本語文字) (英字 | 数字 | "_" | 日本語文字)*

日本語文字 = ひらがな(U+3040〜U+309F)
           | カタカナ(U+30A0〜U+30FF)
           | CJK統合漢字(U+4E00〜U+9FFF)
```

### キーワード

制御構造: `if`, `elseif`, `else`, `endif`, `while`, `endwhile`, `do`, `for`, `endfor`, `return`

論理: `true`, `false`, `not`, `and`, `or`

算術: `mod`

大文字小文字は区別しない（`IF`, `If`, `if` はすべて同じ）。

### 演算子（全角/半角対応）

| 演算子 | 全角 | 半角 | 意味 |
|-------|------|------|------|
| 代入 | `←` | — | 変数に値を代入 |
| 加算 | `＋` | `+` | 加算 / 文字列結合 |
| 減算 | `−` (U+2212), `－` (U+FF0D) | `-` | 減算 |
| 乗算 | `×` | — | 乗算 |
| 除算 | `÷` | — | 除算 |
| 等価 | `＝` | `=` | 等しい |
| 非等価 | `≠` | — | 等しくない |
| 小なり | `＜` | `<` | より小さい |
| 大なり | `＞` | `>` | より大きい |
| 以下 | `≦` | — | 以下 |
| 以上 | `≧` | — | 以上 |
| ビットAND | `∧` | — | 論理積（ビット） |
| ビットOR | `∨` | — | 論理和（ビット） |
| 左シフト | — | `<<` | 左ビットシフト |
| 右シフト | — | `>>` | 右ビットシフト |

### 区切り文字

`(`, `)`, `[`, `]`, `{`, `}`, `,`/`，`, `:`/`：`, `.`

### コメント

```
行コメント     = "//" 任意の文字列 改行
ブロックコメント = "/*" 任意の文字列 "*/"
```

### 宣言マーカー

- `○`: 関数/手続き定義の先頭
- `大域`: 大域（グローバル）変数宣言の先頭

## 型体系

### 基本型

| 型名 | 説明 |
|------|------|
| 整数型 | 整数値 |
| 実数型 | 実数（浮動小数点）値 |
| 文字列型 | 文字の列 |
| 文字型 | 1文字 |
| 論理型 | true / false |

### 配列型

```
型名の配列           # 一次元配列
型名の二次元配列      # 二次元配列
型名配列の配列        # 配列の配列
```

### カスタム型

大文字始まりの英字名はクラス名として扱われる（例: `ListElement`, `PrioQueue`）。

### 型注釈の構文

```
変数宣言   = 型注釈 ":" 変数名 ("," 変数名)* ("←" 式)?
型注釈     = 基本型名
           | 基本型名 "の配列"
           | 基本型名 "の二次元配列"
           | 基本型名 "配列の配列"
           | クラス名
```

## 構文規則

### プログラム構造

```
プログラム = (大域変数宣言 | 関数定義 | 文)*

関数定義   = "○" [型注釈 ":"] 関数名 "(" 引数リスト ")"
             文*

引数リスト = (型注釈 ":" 引数名 ("," 型注釈 ":" 引数名)*)?
```

関数定義は `○` で始まる。戻り値がある場合は型注釈を付ける。戻り値がない（手続き）場合は型注釈を省略する。

```
○整数型: fee(整数型: age)        // 戻り値あり（整数型）
○display(整数型: n)              // 戻り値なし（手続き）
```

### 変数宣言

```
変数宣言     = 型注釈 ":" 変数名リスト ("←" 初期値)?
大域変数宣言 = "大域" ":" 変数宣言
変数名リスト = 変数名 ("," 変数名)*
```

例:
```
整数型: x
整数型: x, y, z
整数型: x ← 0
整数型の配列: arr ← {1, 2, 3}
大域: 整数型: gCount ← 0
```

### 代入文

```
代入文 = 式 "←" 式
```

例:
```
x ← 42
arr[i] ← x + 1
node.next ← newNode
```

### if文

```
if文 = "if" "(" 条件式 ")"
         文*
       ("elseif" "(" 条件式 ")"
         文*)*
       ("else"
         文*)?
       "endif"
```

### while文

```
while文 = "while" "(" 条件式 ")"
            文*
          "endwhile"
```

### do-while文

```
do-while文 = "do"
               文*
             "while" "(" 条件式 ")"
```

`do-while` は最低1回はボディを実行する。

### for文

3つのパターンがある:

**標準パターン（増減）:**
```
for文 = "for" "(" 変数名 "を" 式 "から" 式 "まで" 式 "ずつ" ("増やす"|"減らす") ")"
          文*
        "endfor"
```

例:
```
for (i を 1 から n まで 1 ずつ増やす)
for (i を n から 1 まで 1 ずつ減らす)
for (i を 0 から 10 まで 2 ずつ増やす)
```

**範囲パターン:**
```
for文 = "for" "(" 変数名 "を" 式 "から始めて" 式 "を超えない範囲で" 式 "ずつ" ("増やす"|"減らす") ")"
          文*
        "endfor"
```

**for-each パターン:**
```
for文 = "for" "(" 変数名 "に" 式 "の要素を順に代入する" ")"
          文*
        "endfor"
```

例:
```
for (item に list の要素を順に代入する)
```

### return文

```
return文 = "return" 式?
         | "return" "(" 式 ")"
```

### 日本語アクション文

以下の日本語パターンを文として認識する:

```
末尾追加 = 式 "の末尾に" 式 ("の値")? "を追加する"
値入替   = 式 "と" 式 "の値を入れ替える"
値増減   = 式 "の値を" 数値 ("増やす"|"減らす")
繰返終了 = ("..." "の行から始まる")? "繰返し処理を終了する"
出力     = 式 "を出力する"
複合出力 = 式 "の値と" 式 "の値をこの順に" 区切り種別 "区切りで出力する"
```

### 日本語条件式

条件式は通常の演算子式に加え、以下の日本語パターンを認識する:

```
式 "が" 式 "以下"           →  式 <= 式
式 "が" 式 "以上"           →  式 >= 式
式 "が" 式 "より小さい"     →  式 < 式
式 "が" 式 "より大きい"     →  式 > 式
式 "が" 式 "と等しい"       →  式 == 式
式 "が" 式 "と等しくない"   →  式 != 式
式 "が" "未定義"            →  式 is None
式 "が" "未定義でない"      →  式 is not None
式 "が" 式 "で割り切れる"   →  式 % 式 == 0
式 "が" 式 "でない"         →  式 != 式
```

複合条件は `and` / `or` で結合可能:

```
age が 1 以上 and age が 10 以下
```

## 式の構文

### 演算子優先順位（低→高）

| 優先順位 | 演算子 | 結合性 |
|---------|-------|-------|
| 10 | `or` | 左 |
| 15 | `∨`（ビットOR） | 左 |
| 20 | `and` | 左 |
| 25 | `∧`（ビットAND） | 左 |
| 30 | `＝`, `≠`, `＜`, `＞`, `≦`, `≧` | 左 |
| 35 | `<<`, `>>` | 左 |
| 40 | `＋`, `−` | 左 |
| 50 | `×`, `÷`, `mod` | 左 |
| 60 | `not`, `+`, `-` （単項） | 右 |
| 70 | `.`, `[]`, `()` | 左 |

### アトム（基本式）

```
アトム = 整数リテラル
       | 実数リテラル
       | 文字列リテラル
       | 論理リテラル（true / false）
       | 未定義リテラル
       | 識別子
       | "(" 式 ")"
       | 配列リテラル
```

### 配列リテラル

```
配列リテラル = "{" 式 ("," 式)* "}"
             | "{" 配列リテラル ("," 配列リテラル)* "}"   // ネスト
```

例:
```
{1, 2, 3}
{{1, 2}, {3, 4}}
```

### 後置式

```
配列アクセス   = 式 "[" 式 ("," 式)? "]"
関数呼び出し   = 式 "(" (式 ("," 式)*)? ")"
メンバアクセス = 式 "." 識別子
```

## 別紙1（ITパスポート試験用）と別紙2（FE・AP試験用）の差異

| 機能 | 別紙1（IP） | 別紙2（FE・AP） |
|-----|------------|----------------|
| 関数定義 | ○ | ○ |
| 変数宣言 | ○ | ○ |
| if/elseif/else/endif | ○ | ○ |
| while/endwhile | ○ | ○ |
| do-while | ○ | ○ |
| for/endfor | ○ | ○ |
| 配列 | ○ | ○ |
| メンバアクセス（`.`） | — | ○ |
| クラス/オブジェクト | — | ○ |
| ビット演算 | — | ○ |
| シフト演算 | — | ○ |

本ツールキットは別紙1・別紙2の両方に対応している。
